# AI Bridge Cloud

Cloud relay for the plugin-only AI Bridge architecture.

## Local run

```powershell
python -m pip install -r requirements.txt
uvicorn app:app --reload
```

Required production variables:

- `DATABASE_URL` — supplied by Render PostgreSQL.
- `AI_BRIDGE_API_KEY` — generated by Render; used by the future account/MCP layer.

The plugin registers through `/v1/plugins/register`, displays the returned
six-digit code, and polls `/v1/plugin/poll`. An AI-side integration claims the
code and creates commands after authentication.
